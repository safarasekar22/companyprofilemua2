RASC Makeup - Company Profile Website

Website company profile untuk studio makeup RASC Makeup, dibuat sebagai tugas mata kuliah Web Programming I. Website ini menampilkan informasi seputar studio, layanan yang ditawarkan, serta menyediakan panel admin untuk mengelola data layanan dan tim secara dinamis.

Fitur
- Halaman publik: Beranda, Tentang Kami, Layanan, Kontak (dengan embed lokasi/maps)
- Admin panel dengan sistem login (session-based authentication)
- CRUD data layanan dan anggota tim
- Form kontak untuk pengunjung

Teknologi yang Digunakan
- PHP
- MySQL
- HTML, CSS, Bootstrap
- XAMPP (local server)

Cara Instalasi
1. Clone atau download repository ini
2. Import file database (`.sql`) ke phpMyAdmin
3. Sesuaikan konfigurasi koneksi database (host, username, password, nama database)
4. Pastikan struktur folder `assets/img/` sesuai
5. Jalankan XAMPP/Laragon, aktifkan Apache & MySQL
6. Buka di browser: `localhost/nama-folder-project`
